## üîß Troubleshooting

### If label doesn't print correctly:

**Issue: Label is too small/large**
- **Solution**: Check browser print settings ‚Üí Paper Size ‚Üí Set to "4x6" or "Label"
- **Solution**: Ensure "Scale" is set to 100%, not "Fit to Page"

**Issue: Text is blurry**
- **Solution**: Check Rollo printer darkness settings (increase if needed)
- **Solution**: Ensure printer drivers are up to date

**Issue: Label is cropped**
- **Solution**: Check browser print margins (set to "None" or "Minimum")
- **Solution**: Verify Rollo printer is calibrated (hold green "r" button)

**Issue: Barcode doesn't scan**
- **Solution**: Increase print darkness in Rollo settings
- **Solution**: Ensure label is printed at 100% scale (not scaled)
